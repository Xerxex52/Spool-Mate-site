Common topics

Restore purchases: Open Spool Mate, go to Settings and tap "Restore Purchases". Make sure you are signed in with the same Apple ID used for the original purchase.

iCloud sync: Enable iCloud Sync in Settings and restart the app. Both devices must use the same Apple ID and be connected to the internet.

Export/import: Export your inventory from the Inventory screen (table view) or from Settings. Import a JSON backup from Settings > Import inventory JSON.

Barcode scan: Barcode scanning is part of Spool Mate Pro. The free version supports up to 10 spools with manual and voice entry.
